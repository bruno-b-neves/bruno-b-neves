### 📂 Projetos em Destaque

Abaixo estão alguns dos projetos que desenvolvi para aplicar conceitos de engenharia de software e desenvolvimento full-stack:

#### 🍕 [Project Pizzaria](https://github.com/bruno-b-neves/project_pizzaria)
Sistema completo de gerenciamento para uma pizzaria (Bella Napoli), integrando uma interface moderna com um back-end robusto.
* **Front-End:** Desenvolvido com **Nuxt 3** e **Vue.js**, focado em uma experiência de usuário fluida e responsiva.
* **Back-End:** API construída com **Spring Boot (Java 21)**, utilizando **Spring Data JPA** para persistência de dados.
* **Destaque:** Implementação de regras de negócio complexas, como fechamento de pedidos e cálculos de itens com persistência em banco de dados relacional.

#### 🗄️ [Banco de Dados](https://github.com/bruno-b-neves/Banco_de_Dados)
Repositório dedicado ao estudo e implementação de arquiteturas de dados.
* **Modelagem:** Inclui diagramas Entidade-Relacionamento (ER) e esquemas relacionais para sistemas complexos (como o de adoção de animais).
* **Tecnologias:** Experiência com **MySQL**, focando em integridade referencial, chaves estrangeiras (FK) e otimização de queries.

#### ☕ [Java POO](https://github.com/bruno-b-neves/Java_POO)
Projetos focados nos pilares da Programação Orientada a Objetos utilizando Java.
* **Conceitos Aplicados:** Demonstração prática de **Herança**, **Encapsulamento**, **Polimorfismo** e **Abstração**.
* **Estrutura:** Desenvolvimento de classes bem estruturadas com métodos construtores, getters/setters e lógica de domínio limpa.
